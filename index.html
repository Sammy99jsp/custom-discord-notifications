How to Customise Discord Badges

  1. First, use the following form below. Save the .ico'sto a folder of your choosing.



  2. Close Discord, if it's open.

  3. Copy/Paste the icons from the selected folder into the %appdata%\discord folder.
    Backup
    Note: You may wish to make a backup of the current icons into a New Folder such as tray_backup.

  4. Reload the  Windows Icon Cache

    Run the following under administrator    Command Prompt  
    1. taskkill /IM explorer.exe /F
    2. cd /d %userprofile%\AppData\Local
    3. del IconCache.db /a
    4. explorer.exe

  5. Restart Discord .
